# Traffic Light V2.1.1 更新说明

## V2.1.1 补丁修复

- 修复顶部状态区在“正在检查更新”“检查更新失败”等状态下文字显示不完整的问题。
- 优化按钮和中文状态文字的宽度检查，增强 Windows 100%、125%、150% 缩放下的界面显示稳定性。
- 保持 V2.1 数据目录和安装目录不变，补丁升级不会覆盖用户配置和历史分析结果。

## 新增

- 新增“检查更新”基础系统，支持本地 update.json 模拟官网、远程版本比较、升级包下载、进度显示、取消下载和 SHA256 校验。
- 新增完整静态官网发布端 `release_site`，包含官网首页、latest/update.json、版本固定 update.json、downloads、checksums、本地服务脚本和发布指南。
- 新增报告图片资产机制，每条成功分析记录都会稳定保存对应抽帧图片，并在 Markdown 报告中插入图片和帧信息。
- 新增统一版本配置 `version.json` / `version_info.py`，后续升级只需维护统一版本入口。

## 优化

- 继续使用随包 FFmpeg 作为正式抽帧方案，保留 V2.0 的视频格式与实时流兼容能力。
- 升级下载文件独立存放在 `updates` 目录，不覆盖用户配置和历史分析数据。
- 旧版无图片报告和旧任务档案保持兼容。
